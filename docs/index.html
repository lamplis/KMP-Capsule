üöÄ Overview

KMP-Capsule is a Kotlin Multiplatform port of the excellent Kyant0/Capsule library. It provides perfect capsule shapes for Compose Multiplatform with platform-specific optimizations.

Kotlin Multiplatform Android iOS Compose G2 Continuity
Why KMP-Capsule? While the original Kyant0/Capsule is Android-only, KMP-Capsule extends this excellent work to iOS and other platforms, maintaining the same API while providing platform-optimized implementations.

üì¶ Installation

Add KMP-Capsule as a submodule to your KMP project:

1. Add Submodule

# Clone the repository as a submodule git submodule add https://github.com/lamplis/KMP-Capsule.git libs/KMP-capsule

2. Configure Gradle

// settings.gradle.kts (root) include(":libs:KMP-Capsule") project(":libs:KMP-Capsule").projectDir = File(rootDir, "libs/KMP-capsule/capsule")

3. Add Dependency

// composeApp/build.gradle.kts (commonMain) commonMain.dependencies { implementation(project(":libs:KMP-Capsule")) }

4. Initialize Submodule

# Initialize and update submodules git submodule update --init --recursive

üìñ API Reference

ContinuousCapsule

The main shape provided by KMP-Capsule. It's a platform-agnostic capsule shape that adapts to the container size.

// Import import com.amp_digital.capsule.ContinuousCapsule // Usage val shape: CornerBasedShape = ContinuousCapsule

Properties

  • Type: CornerBasedShape
  • Platform: Android, iOS
  • Behavior: Creates perfect capsule that adapts to container dimensions

Platform Implementations

  • Android: Delegates to Kyant0's ContinuousCapsule for G2 continuity
  • iOS: Uses RoundedCornerShape(percent = 50) for optimal performance

üì± Platform Differences

ü§ñ Android

  • Implementation: Kyant0 ContinuousCapsule
  • Continuity: G2 continuous curves
  • Performance: Optimized B√©zier calculations
  • Features: Advanced curvature profiles
  • Rendering: Hardware accelerated

üçé iOS

  • Implementation: RoundedCornerShape(50%)
  • Continuity: G1 continuous
  • Performance: Native iOS optimization
  • Features: Standard pill shape
  • Rendering: Core Graphics optimized
Note: While Android provides G2 continuity with advanced curvature profiles, iOS uses a simpler but highly optimized implementation. Both provide excellent visual results for typical use cases.

üîß Troubleshooting

Common Issues

Build Errors

Error: "Unresolved reference: ContinuousCapsule"
Solution: Ensure you've added the dependency to commonMain.dependencies and imported com.amp_digital.capsule.ContinuousCapsule

Submodule Issues

Error: Submodule not found
Solution: Run git submodule update --init --recursive to initialize all submodules

Platform-Specific Issues

Android: If you see rendering artifacts, ensure you're using the latest version of Compose
iOS: If shapes appear different, check that you're using the same container dimensions

Debugging Tips

  • Use Modifier.border(1.dp, Color.Red) to visualize shape boundaries
  • Test on both platforms to ensure consistent behavior
  • Check container dimensions - capsules adapt to their container size
  • Verify submodule is pointing to the correct commit
